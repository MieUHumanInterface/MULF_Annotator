  • 注意事項
1.分割をしたい矩形を選択
 矩形の分割は㉙の矩形を選択して垂直線，水平線もしくはその両方を描画することで行っている．
分割したい矩形を選ぶ際には㉙のCanvasに描画した紫色の矩形の左上の番号を⑭で指定して⑫を実行することで，
選択した矩形の拡大画像が⑨のCanvas上に描画される． 2.分割する直線を描画
 垂直に分割したい場合は⑩を，水平に分割したい場合は⑪を押すことで⑨のCanvas上に分割線を表示する．
この垂直線と水平線は一度に一本ずつしか表示させていないので，一度の分割作業で2つないし4つの矩形に分割する．
 3.分割したい位置を選択
 2.で表示させた分割線を分割を行いたい指定の位置に移動させる．
分割線の移動はキーボードのA,W,S,Dを押すことでそれぞれ左,上,下,右に1画素ずつ移動していく．
 4.分割した矩形の情報を出力
 分割線の位置を決定したら，⑫を押すことで③と⑬のテキストボックスにそれぞれ分割した矩形の情報を入力する．
この作業が終われば，㉙のCanvasにも分割した矩形の情報が反映される．
