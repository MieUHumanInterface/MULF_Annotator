1.Block
 Blockの矩形は1つの段落，章といった意味で使用する．
1つのBlockの矩形の中には少なくとも1つのLine,Characterの矩形が必ず存在する．
 2.Line
 Lineの矩形は1つの行と言った意味で使用する．
1つのLineの矩形の中には少なくとも1つのCharacterの矩形が必ず存在する．
 3.Character
 Characterの矩形は1つの文字と言った意味で使用する．

 4.underline機能
 直線（四角枠）に来たら㉘を押し対応しているものに切り替え㉙を押す。 　 線(四角枠)が行途中で終わる場合終了ボタン、次の行に続いている場合継続ボタンを押し、id を同じにする。
