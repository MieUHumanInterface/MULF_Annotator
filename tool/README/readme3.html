  • 注意事項
1.作業はFirefoxでの作業を推奨
 HTMLファイルの動作確認はFirefoxでしか行っていない．そのためその他のブラウザ（Chrome,IE,Safariなど）上で
正しく動作することは保証できない． 2.作業終了後，別の画像ファイルを開く際はページの更新を！
 続けて作業を行いたい場合は一度，ページの更新をして頂きたい．
更新をせずに画像ファイルを開いた場合，前回の画像ファイルが㉙のCanvas上に残ったままになる可能性があるので，
一度更新をして新しく作業をしてもらいたい． 3.画像のサイズが大きいと正しく描画できない
 ウインドウのサイズよりも画像の大きさが大きくなってしまうと正しく描画することが出来ない．
現状での解決策はウインドウサイズを大きくする，ブラウザの表示の縮尺で表示を小さくする，スタイルシートを使用しない設定にする，
といった方法で凌いでいる． 4.作業の修正機能は取り入れていない
 直前もしくは以前の作業をやり直したいという場合の作業を修正できる機能はつけていない．
